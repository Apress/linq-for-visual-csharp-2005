In chapter 2 is present the Poeple.mdf file. You need to open up SQL Server Management Studio and attach the database to your server. This can be accomplished by right-clicking over the Database item in the Object Explorer window and selecting Attach... From the Attach Databases dialog box press the Add... button and browse to the directory where you unzipped the file. Select the people.mdf database and press OK.

To try an example, load the solution into Visual Studio 2005 and uncomment the related Listingx-y region code corresponding to the Listingx-y in the chapter. For example, if you read in chapter 1 of the Join standard query operator and seeing the Listing1-19 snippet code and you are interested in to reproduce the example you have to uncomment the Listing1-19 from the StandardQueryOperator solution within the Chapter01 directory.
